PROJET POUR AIDER LES GENS DE CO-TU À CONSTRUIRE UN ENGRAIS ORGANIQUE MAISON

Fertilisant organique est l'ingrédient qui détermine la qualité et le rendement des cultures. Développer durablement l'agriculture et l'élevage, il est essentiel de produire engrais organique dans le village. Actuellement, le ménage les déchets et les déjections animales n’ont pas été utilisés pour fabriquer engrais. Ces composants sont libérés dans le l’environnement, provoquant une pollution. Les villageois non seulement voulez savoir comment fabriquer de l'engrais organique pour les plantes, mais je veux aussi rendre l’environnement du village plus propre et plus beau.

Le projet vise à aider les villageois à construire une maison produire de l'engrais organique à partir des déjections animales, déchets agricoles et déchets ménagers. Les experts travailleront avec les villageois pour partager les méthodes de classement, transformer et produire des engrais organiques pour leur jardins. L’environnement sera également plus vert, plus propre et plus beau.
